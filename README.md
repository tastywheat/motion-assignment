# Motion Assignment

## Prerequisites
- Node 18+

## Install
```
npm install
```

## Configuration
- Add your Facebook API key
    - Create a `.env` (or make a copy of the `.env.example` file in the root directory)
    - Add your key to the key `FB_KEY`, e.g. `FB_KEY=MY_FB_API_KEY`

## Run
```
npm run clean
npm run build
npm run start
```
